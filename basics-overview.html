Overview

What is the Overcast Network?

The Overcast Network is a Minecraft PvP server started by MonsieurApple and Anxuiz in December 2010. This is a unique team based PvP server with a variety of maps and game modes to play in.

How can I join?

The Overcast Network currently has two main servers, one in Europe, and one in the US. There is also a tournament server used only for competitive tournaments.

US Server: us.oc.tc
EU Server: eu.oc.tc
TM Server: tm.oc.tc

Commands

/join

Usage: /join or /join [team name]
Allows you to join the game by adding you to the team with the least number of players. If you have a donator rank, you can specify the team you want to join.

/toggle

Usage: /toggle [setting]
Toggles the specified setting.

Setting Usage
Death Messages /toggle dm
Join Messages /toggle jm
Private Messages /toggle pm
Punisment Messages /toggle punishments
Observers /toggle obs
Scoreboard /toggle scoreboard
Timer /toggle timer
Blood /toggle blood

/report

Usage: /report [player] [reason]
Reports a rule breaking player to online moderators.

/g

Usage: /g or /g [message]
Sends a message to global chat. If used without arguments, it sets default chat channel to global.

/t

Usage: /t or /t [message]
Sends a message to team chat. If used without arguments, it sets default chat channel to team.

/tell

Usage: /tell [player] [message] or /r [message]
Sents a private message to a specified player. /r sends a message to the player who sent you a message most recently.

/friends

Usage: /friends or /fr
Lists online friends and what servers they are on.

/server

Usage: /server or /server [server]
Returns the server you are currently on. If used with argument, it warps you to the specified server.

/servers

Usage: /servers Lists all online servers as well as maps being played.

/match

Usage: /match
Returns match information, including match time, number of players, score, etc.

/map

Usage: /map
Returns information about map rules, authors, max players etc.

/rot

Usage: /rot or /rot [page number]
Returns information about the map rotations.

/mapnext

Usage: /mapnext or /nm
Returns the next map that will be played.

/staff

Usage: /staff
Returns list of online staff, including which server they are on.
